# Murungaru High School Library Management System

## Overview
This project provides a library management system for Murungaru High School, using a Node.js/Express backend, a React frontend, and a PostgreSQL database.

### Features
- User management (librarians, students, teachers, admins)
- Book inventory for managing available books
- System for checking in and out books
- Reservation system for books
- Fine management for late returns

### Getting Started
1. Clone the repository.
2. Set up the backend and frontend as described in the respective directories.
3. Configure the database settings based on the provided instructions.

### Technologies Used
- Node.js
- Express
- React
- Tailwind CSS
- PostgreSQL
